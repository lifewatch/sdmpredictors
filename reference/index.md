# Package index

## All functions

- [`calculate_statistics()`](http://lifewatch.github.io/sdmpredictors/reference/calculate_statistics.md)
  : Calculate statistics for a given raster.
- [`correlation_groups()`](http://lifewatch.github.io/sdmpredictors/reference/correlation_groups.md)
  : Groups layers based on the Pearson correlation
- [`dataset_citations()`](http://lifewatch.github.io/sdmpredictors/reference/dataset_citations.md)
  : Generate dataset citations
- [`equalareaproj`](http://lifewatch.github.io/sdmpredictors/reference/equalareaproj.md)
  : World Behrmann equal area coordinate reference system (ESRI:54017),
  used when using load_layers with equal_area = TRUE
- [`get_future_layers()`](http://lifewatch.github.io/sdmpredictors/reference/get_future_layers.md)
  : Get the name of future climate layer(s) based on the current climate
  layer(s)
- [`get_layers_info()`](http://lifewatch.github.io/sdmpredictors/reference/get_layers_info.md)
  : Layer info for specific layer codes
- [`get_paleo_layers()`](http://lifewatch.github.io/sdmpredictors/reference/get_paleo_layers.md)
  : Get the name of paleo climate layer(s) based on the current climate
  layer(s)
- [`layer_citations()`](http://lifewatch.github.io/sdmpredictors/reference/layer_citations.md)
  : Generate citations for all layers
- [`layer_stats()`](http://lifewatch.github.io/sdmpredictors/reference/layer_stats.md)
  : Gives basic layer statistics
- [`layers_correlation()`](http://lifewatch.github.io/sdmpredictors/reference/layers_correlation.md)
  : Gives the Pearson correlation between layers
- [`list_datasets()`](http://lifewatch.github.io/sdmpredictors/reference/list_datasets.md)
  : Lists the supported datasets
- [`list_layers()`](http://lifewatch.github.io/sdmpredictors/reference/list_layers.md)
  : List the current climate layers provided by one or more datasets
- [`list_layers_future()`](http://lifewatch.github.io/sdmpredictors/reference/list_layers_future.md)
  : List the future climate layers provided by one or more datasets
- [`list_layers_paleo()`](http://lifewatch.github.io/sdmpredictors/reference/list_layers_paleo.md)
  : List the paleo climate layers provided by one or more datasets
- [`load_layers()`](http://lifewatch.github.io/sdmpredictors/reference/load_layers.md)
  : Load layers
- [`lonlatproj`](http://lifewatch.github.io/sdmpredictors/reference/lonlatproj.md)
  : Longitude/latitude coordinate reference system (EPSG:4326), used
  when using load_layers with equal_area = FALSE
- [`pearson_correlation_matrix()`](http://lifewatch.github.io/sdmpredictors/reference/pearson_correlation_matrix.md)
  : Calculate the Pearson correlation coefficient matrix for a

- [`plot_correlation()`](http://lifewatch.github.io/sdmpredictors/reference/plot_correlation.md)
  : Plot the correlation matrix for the provided layercodes
- [`sdm_to_bo()`](http://lifewatch.github.io/sdmpredictors/reference/sdm_to_bo.md)
  : Translate file name from sdmpredictors to bio-oracle
- [`sdmpredictors`](http://lifewatch.github.io/sdmpredictors/reference/sdmpredictors.md)
  : sdmpredictors: Species Distribution Modeling Predictor Datasets
